# Odin binding for [RtMidi](https://github.com/thestk/rtmidi)

Binding is generated by [karl-zylinski/odin-c-bindgen](https://github.com/karl-zylinski/odin-c-bindgen).

### example
macOS static build
```shellsession
odin build . -extra-linker-flags:"-lc++ -framework CoreMIDI -framework CoreFoundation -framework CoreAudio"
```
